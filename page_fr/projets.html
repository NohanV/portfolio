Mes projets :

Projet Python iCalendar

Ce projet Python a pour objectif d'extraire et de traiter les données des fichiers d'emploi du temps au format .ics, spécifiquement pour les formations universitaires (BUT1, BUT2, BUT3). L'objectif principal est de générer une page HTML organisant les informations des séances d'enseignement. Les données extraites incluent la date, l'heure de début, la durée, le type de séance, la salle, le groupe d'étudiants, et l'enseignant associé.

Projet Podcast Cybersécurité

Ce projet à pour objectif de creer un podcast dans le but d'alerter sur des failles de sécurité. Notre thème est : "Les attaques sur les véhicules autonomes". Ce podcast doit fournir des explications techniques sur la faille de sécurité, fournir des éléments de remédiation face à cette faille, donner des illustrations (cas réels) d'entreprises, de personnes ou d'états impactés par la faille.

Ce projet est actuellement en cours de réalisation.
