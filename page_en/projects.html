My Projects :

Python iCalendar project

The aim of this Python project is to extract and process data from timetable files in .ics format, specifically for university courses (BUT1, BUT2, BUT3). The main objective is to generate an HTML page organizing teaching session information. Extracted data includes date, start time, duration, session type, room, student group and associated teacher.

Cybersecurity Podcast Project

The aim of this project is to create a podcast to warn of security flaws. Our theme is: "Attacks on autonomous vehicles". This podcast should provide technical explanations of the security flaw, provide remediation options for dealing with the flaw, and give illustrations (real cases) of companies, individuals or states affected by the flaw.

This project is currently in progress.
